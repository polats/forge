# Usage
<!-- usage -->
```sh-session
$ npm install -g @polats/forge
$ forge COMMAND
running command...
$ forge (-v|--version|version)
@polats/forge/0.3.1 win32-x64 node-v8.12.0
$ forge --help [COMMAND]
USAGE
  $ forge COMMAND
...
```
